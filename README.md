# Spring Boot API Starter

Production-ready backend REST API built with Java and Spring Boot.

This project demonstrates how to build scalable and maintainable backend systems following Clean Architecture principles. It is designed as a strong foundation for startups and SaaS platforms that need reliable APIs.

## Features

- REST API with Spring Boot
- Clean Architecture (Controller → UseCase → Repository)
- PostgreSQL integration
- Flyway database migrations
- JWT authentication with refresh tokens
- Dockerized development environment
- CI/CD pipeline with GitHub Actions

## Tech Stack

- Java
- Spring Boot
- PostgreSQL
- Docker
- Flyway
- GitHub Actions

## Purpose

This repository demonstrates best practices for building production-ready backend APIs that are scalable, maintainable, and easy to deploy.
